# Trackly 💸

Trackly is a student-friendly finance tracking app designed to help users record expenses, set budgets, view spending history, and understand their spending habits.

## Features

- Add and delete expenses
- Categorize expenses into Food, Transport, Shopping, Entertainment, Savings, and Other
- View total spending
- View category-wise spending
- Set budget limits
- Login and user profile support
- Mobile-friendly interface
- Expense history tracking

## Tech Stack

- React Native
- Expo
- TypeScript
- Express.js
- PostgreSQL
- Drizzle ORM
- Replit
- GitHub

## Project Versions

### Trackly V1
A Python and Streamlit finance tracker web app.

### Trackly V2
A mobile finance tracking app built with React Native and Expo, including authentication, budgets, expenses, and profile screens.

## Why I Built This

I built Trackly as a beginner FinTech project to combine my interest in Computer Science and Finance. The goal is to create a simple finance app that helps students track spending and build better money habits.
